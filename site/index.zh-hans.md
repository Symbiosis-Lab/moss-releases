# 随处写作。随处发布。一切由你掌控。

将你的 Markdown 文件夹变成网站的桌面应用。
右键点击 → 网站即刻上线。

:::buttons
[下载 macOS 版](https://github.com/Symbiosis-Lab/moss-releases/releases/latest)
[文档](docs/)
:::

## 使用方式

:::grid 3
**写作**
用任意编辑器写 Markdown 文件。Obsidian、iA Writer、VS Code，或者纯文本。
---
**右键点击**
在 Finder 中选中你的文件夹，点击"用 moss 发布"。
---
**完成**
网站即刻上线。导航、样式、深色模式——全部自动处理。
:::

## 你的文件，你的网站

文件永远留在你的电脑上。预览无需账号，无需配置服务器。moss 将你的文件夹编译成静态网站，可以部署到任何地方。

## 随处发布

一键部署到 GitHub Pages。将文章同步发布到 [Matters.town](https://matters.town)。为任意页面添加评论功能。这些都是你在设置中按需开启的功能。

## 面向开发者

moss 是一个基于 Tauri v2 的桌面应用，搭载 Rust 编译引擎和 JavaScript 插件系统。CLI 支持自动化流程和 CI 集成。详见[架构说明](docs/architecture.md)和[插件开发](docs/plugin-development.md)。
